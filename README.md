# Snake-Game-
Snake game in c 
## About Game  
This is a simple snake game made using C language 
### Installation
To install 
1. Install c language
2. Clone Repsitory:
```
git clone https://github.com/Saalim398/Snake-Game-.git
```
3. Run the game by typing:
```
main
```
## Instruction
To play use `a s w d` keys on keyboard
